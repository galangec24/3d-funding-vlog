# 3D Funding Vlog Platform

## 🚀 Full-Stack Business Funding Website with 3D Visualization

### Features
- 🎨 Immersive 3D background with Three.js
- ✨ Real-time editable content (Firebase Firestore)
- 🎥 Video vlog gallery (Supabase)
- 💬 Customer support ticket system
- 👑 Admin control panel
- 🔐 Firebase authentication
- 📱 Fully responsive design

### Tech Stack
- **Frontend:** React 18, Three.js, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** Supabase (PostgreSQL)
- **Realtime Content:** Firebase Firestore
- **Auth:** Firebase Authentication
- **Deployment:** Render (backend), Vercel/Netlify (frontend)
